# Project Summary
CleanFixHarish is an innovative business operating platform tailored for service-oriented enterprises, aimed at boosting operational efficiency and enhancing customer engagement. The platform offers a bilingual public website (English and Hebrew), a robust integrated CRM system for effective lead management, and an admin dashboard for real-time operational monitoring. Noteworthy features include WhatsApp integration for improved customer interaction, a streamlined quote request form for lead generation, and a mobile-friendly Progressive Web App (PWA) for seamless user experience across devices.

# Project Module Description
- **Public Website**: Bilingual marketing pages to showcase services and company information.
- **Lead Generation**: Quote request form that generates CRM leads and integrates with WhatsApp for direct communication.
- **Admin CRM Dashboard**: Authenticated panel for managing leads and partners, providing statistics and detailed views.
- **Database Management**: Backend services for executing CRUD operations on various data entities.
- **PWA Features**: Users can install the app on their devices, supported by an "Install App" button and a reminder banner for non-installed users.
- **Contact Information**: Updated contact details including phone, email, and social media links in the footer.

# File Description Inventory
- **app/backend/main.py**: Entry point for backend services.
- **app/frontend/src/App.tsx**: Main React component for the frontend application.
- **app/frontend/src/components/InstallPrompt.tsx**: Prompts users to install the PWA with improved instructions.
- **app/frontend/src/components/Header.tsx**: Contains an effective "Install App" button.
- **app/frontend/src/components/Footer.tsx**: Displays updated contact details and social media links.
- **app/frontend/src/lib/whatsapp.ts**: Manages WhatsApp integration for direct communication.
- **app/backend/services/**: Contains service files for business logic.
- **app/frontend/src/pages/**: React components for various pages in the application.
- **app/backend/data_models/**: JSON files defining structure and initial data for leads, partners, and services.
- **app/frontend/public/assets/logo.png**: Integrated logo used across the application.
- **app/frontend/public/manifest.json**: PWA manifest file defining app properties.
- **app/frontend/public/sw.js**: Service worker for caching and offline support.
- **app/frontend/public/icons/**: Contains resized icons for PWA.

# Technology Stack
- **Frontend**: React 18, TypeScript, Vite, Tailwind CSS
- **Backend**: Atoms Cloud (Auth, Database, Edge Functions)
- **Styling**: Tailwind CSS
- **State Management**: React Context API
- **Routing**: React Router v6
- **PWA Support**: Service Worker, Web App Manifest

# Usage
To install dependencies and set up the project:
1. Navigate to the backend directory and run:
   ```bash
   pip install -r requirements.txt
   ```
2. Navigate to the frontend directory and run:
   ```bash
   pnpm install
   ```
3. Build the frontend:
   ```bash
   pnpm run build
   ```
4. Start the backend service:
   ```bash
   python main.py
   ```
