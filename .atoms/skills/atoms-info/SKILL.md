Atoms Integrations and Tools:
- Mode switching: Atoms now offers Engineer Mode and Team Mode. In Engineer Mode, you activate engineer agent @Alex only. In Team Mode, you invoke multiple agents to collaborate, meeting complex or general requirements. You can switch the model by toggling the button below the input box. See https://help.atoms.dev/en/articles/12129385-mode-switching-guide for more details.
- Atoms Cloud is a Self-Built backend platform that allows users with zero development experience to easily configure a backend for their websites. When creating websites with Atoms, the added data can be stored in the remote database.
- LLM: Atoms offers diverse large language models: Claude Sonnet (default), GPT, Gemini, DeepSeek, and Qwen.
- Upload Files: click the "+Add" button at the bottom left of the input to upload a file or folder.
- Upload Images: select an image by uploading a file or copying the image and paste it directly into the input box.
- Bug Fix function: When website construction encounters issues, a Bug Report appears in the bottom-left corner, prompting users to "Fix all" or click to expand and view specific error details. See https://help.atoms.dev/en/articles/12129264-bug-fix for more details.
- Remix: directly click the remix icon within the conversation area to branch off the current version. Remix helps retain context during development, tracks changes via version summaries, and optimizes resource use. It prevents rapid credit loss from expanded usage or memory growth. To switch between versions, click the "Versions" button on the top-right corner.
- Editor: click the Editor button on the right side bar to view all the file generated by agents. You can also select files and make change here.
- Terminal: click the Terminal button on the right side bar to view agent's executions inside terminals.
- App Viewer is where users can preview application build results before deploy. It provides functions including: a. Replace Elements: Replace elements in App Viewer by selecting them and describing the desired replacement in the input box. b. Add or Exchange Images: After selecting an element in inspect mode, click "add or exchange" to replace or add images. c. Publish Website: Clicking the Publish button opens a dropdown with an editable, publishable new link. You can modify this URL or click Publish directly. See https://help.atoms.dev/en/articles/12129698-app-viewer for more details.
- Share: click the top-right corner to Share your Chat. A dropdown appears where you can click the "Export" button to download all code files, change Chat permissions between Public (anyone can view), Secret (only people with the link can view), or Private (only you can view), copy the share link, or share to social platforms (X, Instagram, LinkedIn, TikTok). see https://help.atoms.dev/en/articles/12129279-share for more details.

Cloud & AI Wallet (for "AI balance is insufficient" / "cloud ai balance" / similar wallet-related questions):
- Cloud & AI Wallet balance is different from plan credits.
- If either AI balance or Cloud balance is exhausted, the user needs to top up the Cloud & AI Wallet to continue the corresponding capability, such as AI features or project publishing.
- Direct them to https://help.atoms.dev/en/articles/14432563-cloud-ai-wallet and tell them they can top up at https://atoms.dev/dashboard?settings=cloudAiBalance.
- If relevant, mention that Free users must upgrade to Pro or Max before they can top up the wallet.
